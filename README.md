## Аннотация
Рассматривается задача формирования нелинейной математической модели движения беспилотного летательного аппарата (БПЛА) самолетного типа с учетом контура стабилизации, необходимой для синтеза системы автоматического управления траекторным движением, в условиях отсутствия необходимых данных для ее построения: аэродинамических, инерционных характеристик БПЛА, данных о двигательной установке и др. Предлагается подход, основанный на аппроксимации динамики БПЛА по полетным данным с использованием сети LSTM - типа рекуррентной нейронной сети, способной учитывать долгосрочные зависимости в последовательных данных и временных рядах. Работа направлена на обеспечение качества траекторного управления в условиях неполноты исходных данных.
## Ключевые слова
LSTM, RNN, аппроксимация, математическая модель движения БПЛА
## Введение

Для решения задач управления летательными аппаратами в первую очередь определяют его математическую модель. Математическая модель должна описывать движение центра масс ЛА в пространстве и его движение вокруг центра масс. Движение ЛА описывается с помощью динамических и кинематических дифференциальных уравнений, которые представляют собой систему нелинейных дифференциальных уравнений, включающую в себя:


## Математическое описание различных моделей аппроксиматоров
В настоящее время существует большое количество аппроксиматоров под самые различные задачи. Если говорить об аппроксимации параметров движения ЛА, изменяющихся во времени,то необходимо использовать модели, которые учитывают порядок данных,т.е. контекст. В работах для решения такой задачи в основном используют:
1) модель в виде передаточной функции с идентификацией МНК;
2) линейная ARX-модель;
3) нелинейная NARX-модель;
4) рекурретная нейронная сеть (RNN);
5) сеть долгой краткосрочной памяти (LSTM).
